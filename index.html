MeshLab open source, portable, and extensible system for the processing and editing of unstructured 3D triangular meshes. For processing raw data from 3D digitization tools/devices and for preparing models for 3D printing. It provides a set of tools for editing, cleaning, healing, inspecting, rendering, texturing and converting meshes.
